# 使用說明

JC  (指令)  [參數1]  [參數2]  [參數3]...
> 每一個字組之間間隔2個空格

### 權限分級
| 等級 | 說明 |
| :---: | :---: |
| 0 | 一般 |
| 1 | 群組總管 |
| 2 | 群組副管 |
| 3 | 機器人總管 |

## 指令表

| 指令 | 中文意義 | 權限等級 | 參數1 | 參數2 | 範例 | 備註 | 快轉 |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: | :---: |

| `Q` | 指定ID範圍的回覆組 | 0 | 起始ID | 終止ID | `JC  Q  11  25` | 一次最多呼叫15組關鍵字 |
| `Q` | 含有指定字的回覆組 | 0 | 部分關鍵字 | | `JC  D  水` | |
| `I` | 指定的回覆組詳細資訊 | 0 | 關鍵字 | | `JC  I  水母` | |

| `SHA` | 獲取SHA224加密字串 | 0 | | | `JC  SHA  水母` | |
